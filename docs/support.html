Frequently Asked Questions

What is the daily message limit?

Free users can send up to 10 messages per day. This limit resets at midnight (local time). To remove this limit, you can configure your own API key in Settings.

How do I configure my own API key?

Go to Settings → AI Engine → Select "Custom Key" → Configure API Key. You can get API keys from providers like OpenAI, Anthropic, Google (Gemini), DeepSeek, or Kimi (Moonshot).

Is my data secure?

Yes! Your chat history is stored locally on your device only. API keys are securely stored in iOS Keychain. We do not have access to your conversations or credentials.

Can I use the app offline?

The app requires an internet connection to generate AI responses. However, you can view your previous chat history offline.

How do I delete my chat history?

You can delete individual conversations by swiping left on them in the chat history view. To delete all data, you can uninstall and reinstall the app.

Why is the AI response slow?

Response time depends on your internet connection and the AI service provider's server load. If using free mode, peak hours may result in longer wait times.

Can I change the app language?

Yes! Go to Settings → Language to switch between English and Simplified Chinese.

How do I report inappropriate AI responses?

Tap the flag icon below any AI message to submit feedback. We take content quality seriously and review all reports.

Troubleshooting

App crashes or freezes

Try these steps:

  • Force close and restart the app
  • Make sure you have the latest version installed
  • Restart your device
  • If the issue persists, contact support

API key not working

Verify that:

  • The API key is entered correctly (no extra spaces)
  • Your API key has sufficient credits/quota
  • You selected the correct provider
  • The API key hasn't expired

Cannot connect to AI service

Check that:

  • You have a stable internet connection
  • VPN is not blocking the connection (if applicable)
  • The AI service is not experiencing downtime
